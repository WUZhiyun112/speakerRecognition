Implementation of an Agent-RL Code Generation System

Framework: VeRL (Versatile RL)   |   Codename: code-r1
Reinforcement Learning LLM Agents Code Generation Self-Correction

Abstract

This project aims to implement code-r1, an advanced agent-based reinforcement learning model specializing in code generation. By leveraging the VeRL framework, we develop a system capable of self-correction through a feedback loop involving a secure execution sandbox and a rule-based reward engine. The core objective is to move beyond static supervised fine-tuning (SFT) toward a dynamic policy that learns from execution errors and environmental feedback.

System Architecture

  • Actor Model: Fine-tuned LLM optimized for reasoning-of-thought (CoT).
  • Environment: Isolated Docker/Subprocess sandbox for code execution.
  • Reward Logic: Multi-tier feedback (Syntax -> Runtime -> Unit Tests).
  • RL Algorithm: PPO/GRPO implementation via VeRL.
